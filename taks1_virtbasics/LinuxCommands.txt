Основные команды Linux
1.	ls
Отображает список файлов и директорий в текущем каталоге.
Примеры: - ls -l — подробный вывод с правами доступа и датами. - ls -a — показывает скрытые файлы (начинаются с точки).
2.	cd Изменяет текущий каталог. Примеры: - cd /path/to/directory — перейти в указанный каталог. - cd .. — перейти на уровень выше.
3.	pwd Показывает полный путь к текущему каталогу.
4.	mkdir Создает новый каталог. Пример: mkdir new_directory.
5.	rmdir
Удаляет пустой каталог.
Пример: rmdir empty_directory.
6.	rm Удаляет файлы или каталоги.
Примеры: - rm file.txt — удалить файл. - rm -r directory — удалить каталог и его содержимое.
7.	cp Копирует файлы и каталоги. Примеры: - cp source.txt destination.txt — копировать файл.
o	cp -r source_directory destination_directory — копировать каталог.
8.	mv Перемещает или переименовывает файлы и каталоги. Пример: mv oldname.txt newname.txt.
9.	touch Создает пустой файл или обновляет временную метку существующего файла.
Пример: touch newfile.txt.
10.	cat Отображает содержимое файла в терминале.
Пример: cat file.txt.
11.	nano / vim / gedit
Редакторы текстовых файлов. Пример: nano file.txt — открыть файл в редакторе nano.
Работа с пакетами
12.	apt (для Debian/Ubuntu) Утилита для управления пакетами. Примеры: - sudo apt update — обновить список доступных пакетов.
o	sudo apt upgrade — обновить установленные пакеты до последних версий. - sudo apt install package_name — установить пакет. - sudo apt remove package_name — удалить пакет. - sudo apt search package_name — поиск пакета по имени.
13.	dpkg (для Debian/Ubuntu)
Утилита для работы с .deb пакетами. Примеры: - sudo dpkg -i package.deb — установить пакет из .deb файла.
o	dpkg -l — показать список установленных пакетов.
14.	yum (для CentOS/RHEL)
Утилита для управления пакетами. Примеры: - sudo yum install package_name — установить пакет. - sudo yum remove package_name — удалить пакет. - sudo yum update — обновить все пакеты.
15.	dnf (для более новых версий CentOS/RHEL)
Замена yum с улучшенной функциональностью.
Примеры: - sudo dnf install package_name — установить пакет.
o	sudo dnf remove package_name — удалить пакет.
o	sudo dnf update — обновить все пакеты.
Другие полезные команды
16.	chmod Изменяет права доступа к файлам и каталогам.
Пример: chmod 755 script.sh.
17.	chown
Изменяет владельца и группу файла или каталога.
Пример: chown user:group file.txt.
18.	find Ищет файлы и каталоги в файловой системе. Пример: find /path/to/search -name "filename".
19.	grep
Ищет строки, соответствующие шаблону в файлах.
Пример: grep "text" file.txt.
20.	df Показывает информацию о доступном и используемом дисковом пространстве.
Пример: df -h (показать в удобном формате).
21.	top
Отображает текущие процессы и их использование ресурсов в реальном времени.
22.	ps
Показывает текущие запущенные процессы. Пример: ps aux (показать все процессы).
23.	kill Завершает процесс по его идентификатору (PID).
Пример: kill 1234.
24.	man Показывает справочную страницу для команды.
Пример: man ls.
